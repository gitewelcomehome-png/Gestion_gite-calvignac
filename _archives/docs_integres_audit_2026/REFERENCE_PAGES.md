# Référence des Pages Principales

## Dashboards

### Dashboard CLIENT (Propriétaires de gîtes)
**Fichier:** `index.html`
- Interface utilisée par les clients SaaS
- Gestion de leurs propres gîtes et réservations
- Section "Vision Globale" avec cartes statistiques

### Dashboard ADMIN (Channel Manager)
**Fichier:** `pages/admin-channel-manager.html`
- Interface de gestion des clients SaaS
- Vue administrateur du système
- Gestion des comptes clients

## Onboarding

### Page d'inscription CLIENT
**Fichier:** `pages/onboarding.html`
- Création de compte pour nouveaux clients
- 4 étapes de collecte d'informations

### Démo d'inscription (test)
**Fichier:** `pages/onboarding-demo.html`
- Prévisualisation sans création de compte
- Accessible depuis admin-channel-manager.html

## Support

### Support CLIENT
**Fichier:** `pages/client-support.html`
- Interface support pour les clients

### Support ADMIN
**Fichier:** `pages/admin-support.html`
- Interface support pour les administrateurs

## Règle Importante
⛔ **Ne JAMAIS modifier index.html sans confirmation explicite** - c'est la page CLIENT en production
