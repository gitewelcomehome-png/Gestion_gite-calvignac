# Archives classées par catégorie

Date de mise en place: 2026-02-24

## Structure

- `sql/` : lots d’archives SQL (cleanup + MVP rebuild)
- `docs/` : lots d’archives documentaires
- `js/` : lots d’archives JavaScript
- `html/` : lots d’archives HTML
- `general/` : lots mixtes historiques (cleanup globaux)

## Règle

Tout nouvel archivage doit être placé dans la catégorie correspondante,
avec un `MANIFEST.md` dans chaque lot archivé.
